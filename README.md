# dotfiles

### Get Started
#### Prerequisites
- curl wget fzf zsh tmux git make gcc g++ bat nodejs npm nvm
- mkdir ~/code  ~/.local/bin
- kitty, nerd fonts
- httpie
- zsh + oh-my-zshell
- zsh-autosuggestions
- zsh-syntax-highlighting
- zsh-vi-mode
 
#### Install
```shell
git clone https://github.com/socoolbear/dotfiles.git ~/.dotfiles

${HOME}/dotfiles/install
```
