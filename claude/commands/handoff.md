1. **대화 전체를 분석**하여 작업 상태를 파악하세요

- **필수 섹션**: 작업 개요, 완료된 작업, 남은 작업, 핵심 참조 파일, 아키텍처 결정 사항, 현재 Git 상태, 다음 세션 시작 프롬프트
